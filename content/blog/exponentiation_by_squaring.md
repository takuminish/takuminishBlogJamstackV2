# 計算量
- O(logn)

# 実装例
```java
  public static long pow(long a, long b) {
    long ans = 1;
    for (long mul = a; b != 0; b >>= 1, mul *= mul) {
        if ((b & 1) != 0) {
            ans *= mul;
        }
    }
    return ans;
  }
```
